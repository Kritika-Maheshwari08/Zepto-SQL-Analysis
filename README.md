# Zepto Inventory & Pricing Analysis using SQL

## Project Overview

This project analyzes Zepto's product inventory, pricing, discounts, and stock availability using MySQL. The goal was to clean raw data, perform exploratory data analysis (EDA), and generate business insights through SQL queries.

---

## Objectives

* Clean and prepare the dataset for analysis.
* Analyze inventory and stock availability.
* Evaluate pricing and discount patterns.
* Identify category-wise trends and business insights.

---

## Dataset Information

The dataset contains product-level information such as:

* Category
* Product Name
* MRP
* Discount Percentage
* Discounted Selling Price
* Available Quantity
* Weight
* Stock Status
